# AJ Revetement — chat assistant

Browser chat UI with a Node backend. Uses the Gemini API; your key stays in `.env` (never committed).

## Setup

1. Clone the repo.
2. `npm install`
3. Copy `.env.example` to `.env` and add `GEMINI_API_KEY` from [Google AI Studio](https://aistudio.google.com/apikey).
4. `npm start` and open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm start` — run the server
- `npm run dev` — same with `--watch` on Node
